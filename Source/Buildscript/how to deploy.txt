Последовательность развёртывания:

- Миграции MS SQL Server (папка Database Migrations)
В конфиге 2Gis.Erm.Migrator.exe.config прописать правильные connection strings (Erm, ErmLogging)
Запустить
2Gis.Erm.Migrator.exe -update

- Миграци ElasticSearch (папка Search Migrations)
В конфиге 2Gis.Erm.Qds.Migrator.exe.config прописать правильные connection strings (Erm, ErmSearch)
Запустить
2Gis.Erm.Qds.Migrator.exe

- Web-приложения и WCF-сервисы (Zip-пакеты)
2Gis.Erm.UI.Web.Mvc.zip
2Gis.Erm.API.WCF.Operations.zip
2Gis.Erm.API.WCF.MoDi.zip
2Gis.Erm.API.WCF.Metadata
2Gis.Erm.API.WCF.OrderValidation
2Gis.Erm.API.WCF.Operations.Special
2Gis.Erm.API.WCF.Releasing

- Сервис выполнения задач по расписанию (Msi-пакет)
2Gis.Erm.TaskService.Installer.msi

- Dynamics CRM (Zip-пакет)
2Gis.Erm.BLCore.MsCRM.HackFiles.zip

- Dynamics CRM (папка Plugin Registration)
Запустить PluginRegistration.exe.
Через UI подключиться к Dynamics CRM
Через UI зарегистрировать 2Gis.Erm.BLCore.MsCRM.Plugins.dll
